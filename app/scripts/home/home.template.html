This is a single page web application for adding, editing, and removing customer records.

Customer section contains implemented features for:

  1. Add a customer

    Following fields are available for entry.
    • Email (required)
    • First Name (required)
    • Last Name (required)
    • Phone Number (optional)
    • Address (optional)

  2. Edit a customer

    Application has been provided a way for editing all the fields for an existing customer.

  3. View a customer

    Application has been provided a way for viewing all the existing customers entered into the system.

  4. Delete a customer

    Application has been provided a way for deleting an existing customer.
